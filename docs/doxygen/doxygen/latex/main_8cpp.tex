\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classCortx}{Cortx}}\textquotesingle{}s Main File. 

This is the main entry point of the Cortex program. It processes the command line arguments, sets up signal handling, and runs the Cortex class. The Cortex class is responsible for the main logic of the program.

The entry point of the program. The only objective here is to get the port number, instantiate the class \mbox{\hyperlink{classCortx}{Cortx}}, to keep it running and handle the signals. 

This function is responsible for the initial setup of the Cortex program. It verifies the command-\/line arguments, handles the system signals, instantiates the \mbox{\hyperlink{classCortx}{Cortx}} class, and runs the main loop of the program.
