Detailed Description

Handle new connections.

This class is responsible for managing the server connections. It starts the server on the specified port and begins listening for new connections. When a new connection is accepted, a new session is created to handle the client-server interaction. Each session is run in its own thread to allow for multiple simultaneous connections.
