cv::Mat Detector::annotate_frame ( const cv::Mat &  frame)

Annotates the detected objects in the frame.

Annotate a frame with detected objects.

This function runs the detection model on the frame and adds bounding boxes and class labels to the objects detected in the frame.

Parameters
frameThe input frame.
Returns
The annotated frame.
Parameters
frame- Frame to be annotated
Returns
Annotated frame

This function takes an input frame, converts it into a blob to feed into the neural network, performs a forward-pass through the network, gets the output and for each detected object, draws bounding boxes and labels on the frame, and then returns the annotated frame.
